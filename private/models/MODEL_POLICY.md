# Model Policy

This is the private model policy for agent routing.

## Goals

- affordability
- safety
- competence
- predictable routing

## Tiers

### Approved hosted

Use for:

- planning
- difficult coding
- high-stakes reasoning
- final synthesis

### Approved local

Use for:

- bulk cleanup
- classification
- first-pass summaries
- repetitive support work
- privacy-sensitive low-risk tasks

### Experimental local-only

Use for:

- cheap experiments
- model comparison
- non-sensitive tasks only

Do not use for:

- sensitive decisions
- trusted final outputs
- autonomous high-impact actions

## Routing rule of thumb

- local first for cheap bulk work
- hosted for hard reasoning and important coding
- experimental only with explicit low-risk scope

## Notes

This policy is private because trust notes and approval status are internal decisions, not public metadata.
