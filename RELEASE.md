# V1 Release Notes

## What this is

A small public repo of AI/computer-useful datasets built from low-risk public metadata sources.

Working public name: `AI Hot Source`

## Why it is worth sending

- It is real, not just an idea.
- It updates automatically.
- It is conservative about terms and access.
- It is immediately useful to agent builders.

## V1 bar

- README explains the value in under 30 seconds.
- Compliance rules are explicit.
- At least two working sources ship with current data.
- Automation is configured.
- Data is easy to inspect in the repo.

## Suggested repo description

Useful public metadata feeds for OpenClaw builders and adjacent agent tooling communities.
